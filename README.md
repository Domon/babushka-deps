# Domon's Babushka Deps

Please see https://babushka.me/running-deps for more information.

## Installation

Install the deps to `~/.babushka/deps`:

    git clone git@github.com:Domon/babushka-deps.git ~/.babushka/deps

## Usage

List available deps:

    babushka list

Run the `rbenv` dep:

    babushka rbenv
